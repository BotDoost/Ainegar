طراحی شیشه‌ای (Glassmorphism)

نمونه طراحی شیشه‌ای

این یک کارت با افکت شیشه‌ای است

چگونه این افکت را ایجاد کنیم؟

برای ایجاد افکت شیشه‌ای از این کد استفاده کنید:

.glass-element {
    background: rgba(255, 255, 255, 0.15);
    backdrop-filter: blur(10px);
    -webkit-backdrop-filter: blur(10px);
    border: 1px solid rgba(255, 255, 255, 0.18);
    box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.37);
}

این تکنیک برای طراحی‌های مدرن و مینیمال بسیار مناسب است.

انیمیشن‌های CSS با clip-path

انیمیشن‌های پیشرفته با clip-path

clip-path به شما امکان می‌دهد اشکال پیچیده ایجاد کنید و بین آن‌ها انیمیشن بسازید:

.shape {
    clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
    transition: clip-path 0.5s ease-in-out;
}

.shape:hover {
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}

این تکنیک برای ایجاد ترنزیشن‌های منحصر به فرد بین المان‌ها عالی است.
